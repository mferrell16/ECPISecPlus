Quick Insert:
Syntax Reference
Basic Hash
Get-FileHash file.txt
Specify Algorithm
Get-FileHash file.txt -Algorithm SHA256 Get-FileHash file.txt -Algorithm MD5
Extract Just Hash Value
(Get-FileHash file.txt).Hash
Store in Variable
$h = (Get-FileHash file.txt).Hash $h # prints value
Compare Two Hashes
$a = (Get-FileHash a.exe).Hash $b = (Get-FileHash b.exe).Hash $a -eq $b
