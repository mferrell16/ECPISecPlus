Exam Traps
WPA2-PSK + STRONG PASSWORD
Sounds secure but uses a shared key — everyone on the network has the same password. Cannot do per-user auth, cannot revoke one user's access, and is vulnerable to offline dictionary attacks if the handshake is captured.
WEP — ALWAYS WRONG
Cryptographically broken since 2001. IV reuse makes it crackable in minutes. Never a correct answer regardless of how the question is worded.
802.1X ≠ JUST WIRELESS
802.1X is a framework that can authenticate wired or wireless clients. It requires a RADIUS server and EAP method. WPA2/WPA3-Enterprise uses 802.1X — they are the same thing described differently.
WPA3 — DICTIONARY ATTACKS
WPA3 uses SAE (Simultaneous Authentication of Equals) replacing PSK. SAE is resistant to offline dictionary attacks because it doesn't expose a handshake that can be captured and brute-forced offline.
